Archives

Start-Log

Description

Starts a new log that records any output from the deployment process.
Note: this uses transcripts, which means it is only be available when running the scripts from the powershell.exe host.
It will simply be ignored when running from Powershell ISE.

Parameters

Name Type Required Description Example
Path string x Folder where the new log file will be created C:\Logs
Name string x Prefix used in the log file name Install_
AppendDate switch If this switch is specified, the current timestamp will be added to the file prefix. Otherwise, the prefix becomes the whole name, and the log will overwrite the same file each time.

Code sample

Start-Log -Path "C:\Logs" -Name "AppX_Deployment_" -AppendDate

Expand-Zip

Description

Extracts the contents of a ZIP archive to the specified folder.

Parameters

Name Type Required Description Example
File string x Path to the ZIP file C:\Builds\Version1.zip
Destination string x Target folder to extract to C:\Target
CleanDestinationFirst switch Add this switch to deleted the destination folder before extracting

Code sample

Expand-Zip -File "C:\Builds\Version1.zip" -Destination "C:\Target"
