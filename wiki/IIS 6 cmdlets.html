IIS 6 cmdlets

This is the list of commandlets for IIS 6 automation.
Each entry details the list of optional and mandatory parameters.

Start-IIS6AppPool

Starts the specified application pool.

Name Type Required Description Example
Name string x Name of the app pool to start MyPool

Code sample

Start-IIS6AppPool -Name "MyPool"

Stop-IIS6AppPool

Stops the specified application pool.

Name Type Required Description Example
Name string x Name of the app pool to stop MyPool

Code sample

Stop-IIS6AppPool -Name "MyPool"

New-IIS6AppPool

Creates an application pool.

Name Type Required Description Example
Name string x Name of the app pool to create MyPool
Username string Identity of the app pool PoolUser
Password string Corresponding password if the user is set Password1

Code sample

New-IIS6AppPool -Name "MyPool" -Username "Bob" -Password "Password1"

Remove-IIS6AppPool

Deletes an application pool.
Note: the application pool has to be empty to be deleted.

Name Type Required Description Example
Name string x Name of the app pool to delete MyPool

Code sample

Remove-IIS6AppPool -Name "MyPool"

Start-IIS6Website

Starts an existing website.

Name Type Required Description Example
Name string x Name of the website to start MySite

Code sample

Start-IIS6Website -Name "MySite"

Stop-IIS6Website

Stops an existing website.

Name Type Required Description Example
Name string x Name of the website to stop MySite

Code sample

Stop-IIS6Website -Name "MySite"

New-IIS6Website

Creates a new website.

Name Type Required Description Example
Name string x Name of the website to create MySite
Path string x Path to the website root on disk C:\Inetpub\MySite
AppPool string x Name of the app pool that will host the website MyPool
Ip string Ip address binding 162.10.16.4
Port integer Port number to use 1234
HostHeader string Host header to use my.website.com
DefaultAccess switch Enables anonymous access to the website
DefaultDoc string List of default documents Default.aspx

Code sample

New-IIS6Website -Name "MySite" -Path "C:\Inetpub\MySite" -AppPool "MyPool" -Ip "162.10.16.4" -Port 1234 -HostHeader "my.website.com" -DefaultAccess -DefaultDoc "Default.aspx"

Remove-IIS6Website

Deletes an existing website.

Name Type Required Description Example
Name string x Name of the website to delete MySite

Code sample

Remove-IIS6Website -Name "MySite"

New-IIS6VirtualDirectory

Creates a new virtual directory inside a given website.

Name Type Required Description Example
Website string x Name of the website that will contain the virtual directory MySite
Name string x Virtual directory name Content
Path string x Path the directory should point to C:\Inetpub\Content

Code sample

New-IIS6VirtualDirectory -Website "MySite" -Name "Content" -Path "C:\Inetpub\Content"

Remove-IIS6VirtualDirectory

Deletes an existing virtual directory from a website.

Name Type Required Description Example
Website string x Name of the website that contains the virtual directory MySite
Name string x Name of the virtual directory to delete Content

Code sample

Remove-IIS6VirtualDirectory -Website "MySite" -Name "Content"
